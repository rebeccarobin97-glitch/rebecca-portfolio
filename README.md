# Rebecca Robin Portfolio

A simple static portfolio website for job applications.

## How to publish on GitHub Pages
1. Create a new public GitHub repository named `rebecca-portfolio`.
2. Upload everything in this folder.
3. Go to Settings → Pages.
4. Under Branch, select `main` and `/root`, then save.
5. Wait a minute or two for GitHub to generate your live URL.

## Before publishing
Open `index.html` and replace:
- `your-email@example.com`
- The placeholder LinkedIn URL
- Add your resume PDF to `assets/pdfs/` if you want a resume button.
